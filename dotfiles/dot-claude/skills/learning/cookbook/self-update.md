# Self-Update Cookbook

Update the learning skill's reference files with the latest pedagogical research and learning methodology insights.

## When to Run

- Periodically (monthly) to stay current with learning science
- When the user reports that generated plans feel outdated
- Invoke via: `/learning self-update`

## Sources

| Source | URL | Updates |
|--------|-----|---------|
| Learn Anything | https://learn-anything.xyz | Learning path curation, new topics |
| OpenClassrooms - Learn How to Learn | https://openclassrooms.com/en/courses/5281811-learn-how-to-learn | Meta-learning techniques |
| Reddit GetStudying | https://www.reddit.com/r/GetStudying/top/?t=month | Community learning tips |

## Update Procedure

1. **Fetch sources**: Use WebFetch/WebSearch to check each source for new insights
2. **Compare**: Read current PEDAGOGY.md and ASSESSMENT.md
3. **Identify gaps**: New techniques, updated best practices, new exercise types
4. **Propose changes**: List what would be added/modified
5. **Confirm with user**: Present proposed changes before applying
6. **Apply**: Edit PEDAGOGY.md and/or ASSESSMENT.md with new content
7. **Version bump**: Update SKILL.md frontmatter version

## What to Look For

- New evidence-based learning techniques
- Updated spaced repetition intervals
- New exercise formats (especially interactive ones)
- Changes to assessment methodologies
- New resource platforms or learning tools
- Domain-specific pedagogical advances

## What NOT to Change

- The core phase structure (Foundation/Development/Mastery)
- The SMART objectives template
- The Obsidian output format (PLAN-FORMAT.md)
- The assessment scoring rubric (unless research strongly supports a change)

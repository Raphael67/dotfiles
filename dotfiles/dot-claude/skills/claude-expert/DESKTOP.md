# Claude Desktop App

Reference for Claude Desktop application features and capabilities.

> This file is auto-maintained by the self-update workflow. It will be populated with Desktop-specific features on the next `/claude-expert self-update` run.

## Overview

Claude Desktop is the native macOS/Windows app for interacting with Claude. It shares the claude.ai web experience but adds native OS integration.

## Key Features

### Conversations
- Standard chat interface with Claude models
- File and image uploads
- Project-based conversations with persistent context

### MCP Integration
- Desktop supports MCP servers configured in `~/Library/Application Support/Claude/claude_desktop_config.json`
- Servers run as local processes managed by the Desktop app

### Scheduled Tasks
- Schedule prompts to run automatically at set intervals
- Useful for recurring checks, reports, and monitoring

### Cowork Mode
- Collaborative mode for working alongside Claude on longer tasks
- Persistent session with shared context
